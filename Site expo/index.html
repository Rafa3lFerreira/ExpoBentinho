O que é uma API

API significa "Application Programming Interface". Uma API é um conjunto de regras e protocolos permitindo assim diferentes programas se comunicarem entre si.

São alguns exemplos:

• Comunição entre aplicativos;
• Maior segurança no programa;
• Padroes e documentações;
• Economização de esforços.

Diferença entre ambos

Constantementes são usados juntos, porém são conceitos diferentes.

As principais diferenças entre eles:

Single Aplicaiton

• Tipo de Aplicação;
• Funcionalidade de carregar uma pagina html e atualiza-la;
• Exemplos: E-commerce e rede sociais usam SPAs.

APIs

• Tipo de Interface de programação;
• Funcionalidade de busca ou enviar dados;
• Exemplos: Desenvolvimento web interagindo com processamento de pagamentos.

O que é Single Aplication

Single Application ou Single Page Application (SPA) é um termo usado na programação web para descrever um tipo específico para mostrar uma pagina interativa que funciona como unica

São algumas caracteristicas:

• Interação com APIs;
• Atualizações do conteudo sem recaregar a pagina inteira;
• Melhor interativade na página da SPA;
• Oferece melhor experiência ao usuario.
